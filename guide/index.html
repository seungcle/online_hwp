HWP 온라인 편집기
사용 가이드 자주 묻는 질문 서비스 소개

사용 가이드

HWP 온라인 편집기는 별도 설치 없이 브라우저에서 바로 HWP, HWPX 파일을 열고 편집할 수 있는 무료 서비스입니다.

1 파일 열기

HWP 또는 HWPX 파일을 편집기에서 여는 방법은 두 가지입니다.

  • 드래그 앤 드롭: 파일을 화면 중앙의 업로드 영역에 끌어다 놓으세요.
  • 파일 선택: 「파일 선택」 버튼을 클릭해 탐색기에서 파일을 선택하세요.
💡 .hwp와 .hwpx 두 형식 모두 지원합니다.

2 문서 편집

파일이 열리면 편집기가 자동으로 실행됩니다. 텍스트 수정, 서식 변경 등 다양한 편집 작업을 수행할 수 있습니다. 상단 툴바를 통해 여러 편집 도구를 사용할 수 있습니다.

  • 텍스트 입력 및 수정
  • 글꼴, 크기, 색상 등 서식 변경
  • 표, 이미지 삽입

3 HWP로 저장

편집이 완료된 문서를 HWP 파일로 저장하려면 상단 우측의 「저장」 버튼을 클릭하세요. 파일이 자동으로 다운로드됩니다.

💡 HWPX 파일을 열었더라도 저장 시 .hwp 형식으로 변환되어 다운로드됩니다.

4 PDF로 내보내기

문서를 PDF 형식으로 내보내려면 상단의 「PDF」 버튼을 클릭하세요. 브라우저의 인쇄 기능을 통해 PDF로 저장할 수 있습니다.

  • 「PDF」 버튼 클릭 → 인쇄 다이얼로그 열림
  • 대상(프린터)에서 「PDF로 저장」 또는 「Microsoft Print to PDF」 선택
  • 「저장」 클릭

5 새 문서 시작

상단의 「새 문서」 버튼을 클릭하면 빈 문서로 새 작업을 시작할 수 있습니다.
